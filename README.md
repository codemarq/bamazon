# Bamazon node.js app
Bamazon is a Node JS app which mimics an Amazon type of store.  This app is written in Node.JS and uses a local MySQL database.

## Technologies Used
* JavaScript
* Node.JS
* MySQL
* npm dependencies:
	* inquirer
	* mysql
	* cli-table
